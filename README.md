# SwipeMenuRecyclerView
滑动抽屉的RecyclerView实现版本，抽屉效果可选
1.Menu 不跟随滑动，停留在下层
2.Menu 跟随滑动


##### 使用
```
buildscript {
    repositories {
        jcenter()
    }
    dependencies {
        ...
        classpath 'com.magicyoung.swiperecyclerview:1.0.0'
    }
}
```
